If the line height of a paragraph is too tight, the reader’s eye gets trapped, accidentally reading the same line twice. If it is too loose, the text falls apart and loses its structure. By default, browsers use a unitless multiplier for text, but you need to dial in the absolute sweet spot for long-form reading.
